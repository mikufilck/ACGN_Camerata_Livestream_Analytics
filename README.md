# ACGN_Camerata_Livestream_Analytics
ACGN同好会直播数据分析工具

# 如何使用
这个工具是用来配合神奇弹幕机器人生成的.log和.csv文件使用的，它并不具备采集功能

在导入数据时，log文件需要打包放在文件夹中，会自动遍历选中的文件夹导入其中的所有log文件，csv文件需要选中单一文件

# 它能做什么？
为直播公会和主播提供一个数据分析的工具，本工具已经完成了以下内容：

## 对于主播

* 直播热力图：根据数据制作热力图，让你知道你的直播间什么时候最活跃，你可以通过这个优化你的直播时间
* 月度韦恩图：对比本月与上月的粉丝，看你的留存率如何
* 核心粉丝排名与散点图：直观的分类你的粉丝群体
* 场次分析：包含单场四线折线图（进房，弹幕，关注，流水）和当场的词云图
* 周/月分析：每周/每月的进房和弹幕折线图
* 场/周/月对比：对比不同场次/周/月份，让你精准确定哪个时间出现问题
* 礼物团块图：列出你一段时间内不同礼物的收入总和并进行对比，这可能可以缓解低舰长但是高礼物收入的主播的焦虑
* 流量转化漏斗图：月度韦恩图的升级版，如果你希望知道更细致的内容（谁不仅留存了，还消费了，谁变成了常在的核心粉丝），那这就是你想要的
* 热度散点图：这个图可以让你知道你的话题与你的营收是否强相关

## 对于运营

* 多主播场次分析：分析不同主播的表现如何
* 多主播韦恩图：让你知道这几位主播的粉丝重合度如何，这几位主播之间能互补还是内卷，并以此调整运营策略

# 发展计划
目前只是一个简单的分析工具，有些图表的实际效用可能没有那么大，需要通过实践反馈进行修正并新增其他更有用的图表
有计划之后变成一个集合采集+分析的一体化工具
